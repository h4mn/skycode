# Experiments

Protótipos e experimentos isolados do projeto SkyCode.

## Diretrizes

- Manter experimentos isolados do código principal
- Documentar objetivos e resultados de cada experimento
- Limpar experimentos concluídos ou abandonados

## Estrutura Sugerida

```
experiments/
├── experimento-nome/
│   ├── README.md
│   ├── src/
│   └── results/
```

## Exemplos de Experimentos

- Novos pontos de integração com IA
- Protótipos de UI/UX
- Testes de performance
- Validação de conceitos técnicos
