# Skybridge Integration

Módulos de integração de IA com o VS Code.

## Estrutura Esperada

```
skybridge-integration/
├── core/              # Core da integração Skybridge
├── extensions/        # Extensões customizadas
├── themes/           # Temas personalizados
└── api/              # Interfaces de API para IA
```

## Diretrizes

- Manter módulos independentes do core do VS Code
- Desenvolver integrações como extensões separadas
- Documentar todos os pontos de integração
