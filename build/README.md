# Build Outputs

Diretório para saídas de build do projeto SkyCode.

## Estrutura

```
build/
├── vscode/           # Builds do VS Code modificados
├── extensions/       # Extensões compiladas
├── packages/         # Pacotes para distribuição
└── logs/            # Logs de build
```

## Observações

- Este diretório deve ser limpo periodicamente
- Não commitar arquivos de build no versionamento
- Mantém artefatos de desenvolvimento isolados
